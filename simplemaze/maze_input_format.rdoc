== Format for the Maze input files
As outlined below

    <height>
    <width>
    <number_info>
    ...
    height lines, with width nodes
    ...
    num_info lines, with
    <row> <col> <info_word>
    ...


(Rows start numbered at 0 at the top and go down.)
(Columns start numbered at 0 at the left and go right.)
